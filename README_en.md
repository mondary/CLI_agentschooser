# CLI Agents Chooser

![Project icon](icon.png)

[🇬🇧 EN](README_en.md) · [🇫🇷 FR](README.md)

✨ A simple and fast menu to launch the CLI agent of your choice.

## ✅ Features
- Centralize your agents in a single entry point.
- Avoid memorizing all commands.
- Works locally without external dependencies.

## 🧠 Usage
```bash
./src/choose_agent.sh
```

## ⚙️ Settings
Open `src/choose_agent.sh` to modify:
- The menu text.
- The launched commands.
- The colors.

## 🧾 Commands
Run the script directly in your terminal. An alias is recommended:
```bash
alias agent="~/Documents/GitHub/PROJECTS/CLI_Agentschooser/src/choose_agent.sh"
```

## 📦 Build & Package
No build required. Ready-to-use versions are in `/release`.

## 🧪 Install
```bash
chmod +x src/choose_agent.sh
```
